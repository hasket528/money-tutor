一、教材定位與適用對象

金錢小達人是一套針對認知功能輕度至中度障礙學生設計的金錢教育互動教材，涵蓋「數學基礎 → 貨幣認知 → 生活應用 → 預算規劃 → 溝通泛化」完整學習路徑。適用於國小至國中特教班、資源班之個別化教學與小組教學，亦可供家長在家延伸練習。

設計原則
大按鈕與高對比介面、全程語音支持（台灣中文）、立即回饋、無錯學習（errorless learning）、系統性提示褪除、可調難度與題數。
環境需求
電腦或平板＋Edge / Chrome 瀏覽器即可，無需安裝軟體。語音以 Microsoft 雅婷（線上）效果最佳；購物對話練習支援離線使用（PWA）。

系統組成

📚 24 個學習單元（F/C/A/B 四系列） 🐕 購物對話練習（金好聊） 🎁 全站共用獎勵系統 📝 作業單產生器（可印/PDF） 🕵️ 遊戲室 🦉 金錢故事冒險

一之二、設計理念：實證教學法（EBP）的系統化實作

本教材的每個功能都對應特殊教育實證教學法，並以軟體將其「自動化、一致化」：

系統功能對應教學理論實作方式
🪙 金幣 → 💎 寶石 → 🐷 寵物/寶物代幣制度（Token Economy）三層增強：金幣＝努力（完成任務即得）、寶石＝品質（🌟全程無錯才得）、寵物進化與寶物＝後援增強物（backup reinforcer），將立即增強與延宕滿足串成系統
提示褪除 L4→L1（購物練習）系統性提示褪除（Prompt Fading）完整示範→句框填空→詞庫組句→獨立表達，依連續答對自動升級；精熟標準可調（ABA 精熟原則）
L4 答錯直接示範、選項模式正解高亮無錯學習（Errorless Learning）降低錯誤練習的固著；🌟無錯通過另予品質增強
A 系列步驟化流程＋輔助點擊工作分析（Task Analysis）販賣機/點餐/ATM 拆解為可教學的步驟鏈，逐步褪除輔助
真幣圖像、模擬情境、實地延伸CRA 序列與泛化設計具體（實物教具卡）→ 表徵（擬真模擬）→ 抽象（數字計算）；教案含社區實地任務促進類化
學習歷程＋錯誤嘗試＋精熟標記資料本位決策（Data-Based Decision Making）自動蒐集正確率/錯誤嘗試/時間，教師依數據調整目標（IEP 檢核）
😀😐😣 學生自評自我決策／後設認知（Self-Determination）每次練習後自評，教師觀察「自評與實際表現落差」
🧭 能力快篩起點行為評估（課程本位評量 CBA）12 題技能階梯，最早失誤處＝教學起點；期初期末重測＝前後測

三、購物對話練習（金好聊）— 溝通泛化模組

以 10 個社區場所（便利商店、早餐店、超市、藥局、夜市攤販、公車、捷運⋯）、200+ 對話步驟，訓練學生「開口買東西」的功能性溝通。對應課綱：社會技巧：基本溝通與互動語文：口語表達（調整）

三種反應模式
🎤 說話（語音辨識）／🔘 選項點選／⌨️ 打字——依學生口語能力選擇，同一教材三種近端目標。
四級提示鷹架（自動褪除）
L4 完整示範 → L3 句框填空 → L2 詞庫組句 → L1 獨立表達。連續答對自動升級（精熟標準可調），支持無錯學習。
IEP 數據
自動記錄每位學生的正確率、提示等級、嘗試次數；一鍵列印個別學習報告與班級總覽，直接作為 IEP 檢核佐證。
學生名冊共用
與獎勵系統共用名冊；完成練習自動累積獎勵分數，增強系統一體化。

四、教師自編工具：對話工作室 🎙️（本教材特色）

老師不需任何技術背景，即可自編貼近班級生活的對話教材：

  1. 購物練習首頁點「➕ 自訂情境」，輸入情境名稱（例：合作社買午餐）。
  2. 逐步填入「店員說的話」與「學生的標準答案」，錯誤選項可留空自動補齊。
  3. 每一步可按 🎙️ 錄音 錄下老師自己的聲音（學生對熟悉聲音反應更佳）；不錄則由電腦語音自動朗讀。
  4. 儲存後立即出現在首頁，三種模式與四級鷹架全部自動適用。
  5. 到「設定 → 對話包分享」可將自編情境連同錄音匯出成單一檔案，分享給其他老師匯入使用。

自編情境與錄音都儲存在瀏覽器本機，不需網路帳號；換電腦請使用「對話包」匯出／匯入搬移。

四之二、金隊長成長生態系 🤖（評量・動機・增強的整合系統）

主頁的 金隊長（AI 成長管理機器人）整合了特教實證教學法中的「代幣制度（token economy）」與資料本位決策：

🧭 能力快篩（起點評估）
12 題階梯式快篩（數概念→貨幣認知），最早失誤處＝建議學習起點，直接對應「起點行為評估」流程；結果自動寫入學習歷程。開學前測、期末後測即成前後測設計。
🚀 每日任務＋⭐ 老師指派
系統每天自動出 3 個任務（自動瞄準正確率最低的弱點單元）；老師另可於「老師指派區」為個別學生指定單元（差異化教學）。完成領金幣。
🪙💎 三層代幣
金幣＝努力（完成任務）、寶石＝品質（🌟無錯通過才有）、寶物／寵物＝後援增強物。學生可在「學生成長頁」用金幣餵養存錢寵物（金錢豬），從神祕金蛋一路進化成黃金飛天豬。
🎯 精熟判準＋😀 學生自評
老師可設定精熟標準（正確率×連續次數，全班預設或個別覆寫），歷程自動標 ✅精熟/🔶接近/⬜未達；每次練習後學生自評（😀😐😣），教師可觀察「自評與實際表現的落差」＝後設認知指標。

📦 資料保全：所有資料存於瀏覽器本機。請定期到金隊長「備份全站資料」匯出單一備份檔（含名冊、紀錄、錄音、寵物）；換電腦用「還原備份」即可搬家。

五、獎勵系統與作業單

🎁 獎勵系統（全站共用）
建立學生名冊（可放照片）→ 各單元完成挑戰自動送分 → 累積分數兌換班級增強物。支援自訂加分圖示與音效；主站與購物練習共用同一份名冊與分數。
📝 作業單產生器
各單元設定頁可一鍵產生對應紙本作業單（含金錢圖示），支援難度與題數調整、直接列印或下載 PDF，作為課後評量與家庭練習。

📊 學習歷程總覽（全站）

24 個單元與購物對話練習的完成紀錄（分數、時間、學生）都會自動存入同一份本機學習資料庫， 開啟 學習歷程總覽 即可檢視全部練習紀錄、匯出 CSV 或列印，作為 IEP 佐證資料。

建議教學流程

  1. 起點評估：以 F、C 系列確認數概念與貨幣認知起點。
  2. 情境教學：依 IEP 目標選擇 A 系列單元，先「輔助點擊模式」再褪除為普通模式。
  3. 素養延伸：B 系列建立預算與儲蓄概念。
  4. 溝通泛化：購物對話練習，從選項模式進階到說話模式，再到真實社區情境演練。
  5. 評量與增強：作業單檢核學習成效；獎勵系統維持動機；學習報告佐證 IEP。

六、資料與隱私聲明

本教材為純前端網頁：學生姓名、照片、成績、錄音全部只儲存在該台電腦的瀏覽器中（localStorage / IndexedDB），不會上傳到任何伺服器。清除瀏覽器資料會一併清除上述紀錄，重要資料請以「學習資料備份」與「對話包分享」功能定期匯出保存。
